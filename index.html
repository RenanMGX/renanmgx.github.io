Projetos

Telemetria

Python Repositorio Git

Este projeto tem a finalidade de coletar dados dos computadores na rede e enviá-los para um servidor que irá salvá-los em um arquivo .csv. Posteriormente, esses dados serão carregados em um Power BI para exibi-los em gráficos relativos aos computadores da rede.
Este projeto está dividido em 3 scripts:
1 - É executado no computador e a cada 5 minutos coleta os dados e os envia para o servidor usando um arquivo .json.
2 - É um script executado no servidor por meio de uma aplicação Flask que coleta os dados dos computadores e os salva em um arquivo .csv.
3 - É um script projetado para ser executado via GPO (Group Policy Object) que instala o 1º script na máquina do usuário, permitindo que ele inicie automaticamente junto com o Windows.

bot_whatsapp

Python Repositorio Git

Este script lê uma planilha do Excel, salvando o número de telefone, mensagem e caminho de um arquivo.
Em seguida, o script dispara mensagens no WhatsApp de forma sequencial para todos os contatos salvos na planilha. No final, uma interface gráfica informa quais mensagens foram entregues.

quebra_excel

Python Privado

Este script acessa uma planilha do Excel que contém vários dados misturados. O script separa os dados por setor e cria uma nova planilha com cada conjunto de dados separado. Utilizei multiprocess e criei uma fórmula que identifica todos os núcleos do computador e separa 90% deles para o script, permitindo o processamento de múltiplas planilhas ao mesmo tempo.
Anteriormente, esse processo era realizado manualmente uma vez por mês e levava cerca de um dia e meio, enquanto com o script, leva aproximadamente 20 minutos.

bot_cotar_moeda

Python Repositorio Git

Este script, usando o Selenium, acessa o site do Banco do Brasil e, a pedido do cliente, coleta o valor da cotação do dólar e do euro sempre do dia anterior. Ele salva esses dados em uma variável e, em seguida, abre o programa SAP para registrar a cotação das moedas em uma transação do SAP.
O script é executado todos os dias às 7:00 da manhã, e foram tratados os erros de forma a minimizar a necessidade de manutenção. Ele identifica quando não houve cotação no dia anterior e, se ficar sem internet ou se o site estiver fora do ar, ele faz uma pausa e tenta novamente algumas vezes, com intervalos de 30 minutos. Caso não consiga abrir o programa SAP ou salvar a transação, encerra suas atividades e fica pronto para a próxima execução. Todas as informações, incluindo problemas e acertos do script, são registradas em um arquivo de log

bot_instalador

Este script abre uma interface gráfica listando alguns programas para instalar no computador. Os programas estão em uma pasta compartilhada na rede e são executados pelo script. Alocação de Resultados

Python Privado

Este script é extenso, contendo mais de 800 linhas, e nem todos os detalhes podem ser revelados.
O script possui uma interface gráfica para orientação do usuário. Ele acessa várias planilhas do Excel, tratando e filtrando os dados, em seguida, os salva em arquivos JSON para serem carregados posteriormente como uma base de dados. O usuário importa outra planilha do Excel e, com base nos dados já salvos, aplica regras extensas de alocação de contas. Posteriormente, o script salva uma nova planilha com os dados tratados.
Este script ainda está em fase de teste, porém, está substituindo um processo manual que era realizado mensalmente e que costumava levar até 7 dias. Com o uso do script, o processo agora leva cerca de 20 minutos.

Bot_imobme

Python Privado

Este script utiliza o Selenium para acessar o site do Imobme, realizar o login como um usuário e navegar até a opção de geração de relatórios. A geração de relatórios pode ser demorada, então o script aguarda a conclusão e, em seguida, salva o arquivo em uma pasta específica. O arquivo é salvo por padrão no formato Excel, mas, a pedido do usuário, o script o converte em um arquivo .csv separado por ";".
O script está aguardando aprovação para iniciar os testes no ambiente de produção.
